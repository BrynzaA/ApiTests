## Тесты создания ресурсов
### ID: TC-11
Название: Создание папки с минимальными обязательными полями через REST API
Предусловия:
Пользователь авторизован с валидным OAuth-токеном
Эндпоинт /vt/disk/resources доступен

Шаги:

Сгенерировать уникальный путь для папки
Сформировать POST-запрос с параметром path
Отправить PUT-запрос на создание папки
Зафиксировать код и тело ответа

Тестовые данные: /test_folder_${timestamp}

Ожидаемый результат:

Статус-код ответа: 201 Created
В теле ответа присутствуют поля: method, href, templated
Заголовок Content-Type: application/json

### ID: TC-12
Название: Создание вложенной папки через REST API
Предусловия:
Пользователь авторизован
Родительская папка существует

Шаги:

Создать родительскую папку /parent_folder
Сформировать PUT-запрос с параметром path=/parent_folder/child_folder
Отправить запрос на создание папки
Зафиксировать ответ

Тестовые данные: /parent_folder/child_folder

Ожидаемый результат:
Статус-код ответа: 201 Created
Папка успешно создана в указанном пути

## Тесты удаления ресурсов
### ID: TC-21
Название: Удаление папки с перемещением в Корзину
Предусловия:
Создана папка с известным путем
Папка существует и доступна

Шаги:
Убедиться, что папка доступна через GET-запрос
Отправить DELETE-запрос на эндпоинт /v1/disk/resources с параметром path
Зафиксировать ответ API
Проверить наличие папки в Корзине

Тестовые данные: path=/test_folder_to_delete

Ожидаемый результат:
Ответ на DELETE-запрос: статус-код 204 (пустое тело)
Папка перемещена в Корзину
Исходная папка недоступна в основном хранилище

### ID: TC-22
Название: Безвозвратное удаление папки (permanently=true)
Предусловия:
Создана папка с известным путем

Шаги:
Отправить DELETE-запрос с параметрами path и permanently=true
Зафиксировать ответ API
Проверить отсутствие папки в основном хранилище
Проверить отсутствие папки в Корзине

Тестовые данные: path=/test_folder_permanent, permanently=true
Ожидаемый результат:

Ответ на DELETE-запрос: статус-код 204
Папка отсутствует в основном хранилище (GET запрос возвращает 404)
Папка отсутствует в Корзине
